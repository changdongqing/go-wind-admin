# 物模型属性单位清单

> 参考标准：GB 3100-1993《国际单位制及其应用》、GB/T 35102-2017《物联网 感知控制设备描述规范》、阿里云/华为云/OneNET 物联网平台物模型惯例、SI 国际单位制。

## 换算公式说明

- `x` 代表原单位的数值，公式结果为基准单位的数值
- 例：cm → m 的换算公式 `x*0.01`，即 5cm = 5×0.01 = 0.05m
- 温度为特殊偏移换算，非纯乘法
- 对数单位（dB、dBm）不可线性换算
- 浓度 ppm/ppb 与 mg/m³ 之间无固定换算系数，依赖气体分子量与温度

## 补充说明

1. **温度换算特殊**：温度是唯一需要偏移量换算的物理量。℃→K 为 `x+273.15`，非纯乘法。若仅做温差换算（如温升 ΔT），则 ℃与K等价（ΔT: 1℃ = 1K），℉→K温差换算为 `x*5/9`。

2. **对数单位（dB、dBm、dB(A)）**：分贝类单位是对数尺度，不能线性换算。dBm→mW 的换算为 `10^(x/10)` mW。

3. **等量纲单位（VA、var、kvarh）**：视在功率(VA)、无功功率(var)与有功功率(W)量纲相同但物理含义不同，不可直接等价换算。kvarh与kWh量纲相同但分别计量无功电能和有功电能。

4. **浓度 ppm/ppb 与 mg/m³**：两者之间无固定换算系数，依赖气体分子量(M)和温度(T)。近似公式：`mg/m³ = ppm × M / 22.4`（标准状况 0°C, 101.325kPa），或 `mg/m³ = ppm × M / 24.45`（25°C, 101.325kPa）。

5. **中国法定计量单位**：依据 GB 3100-1993，中国法定计量单位包括 SI 单位、SI 词头组合单位，以及国家选定的非 SI 单位（min、h、d、°、′、″、L、t、ha、rpm、dB、nmi 等）。"亩"为非法定但广泛使用的传统面积单位。

6. **物联网平台惯例**：阿里云 TSL、华为云 IoTDA、OneNET 等平台属性定义中单位为自由文本字段，常见使用 ℃、%RH、Pa、kPa、MPa、V、A、kW、kWh、Hz、rpm、m³/h、L/min、ppm、lx、dBm 等。
