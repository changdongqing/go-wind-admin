# 物模型特征清单

> 依据标准：GB/T 35102-2017《物联网 感知控制设备描述规范》、GB/T 37044-2018《物联网 信息交换和共享》、阿里云物联网平台TSL规范、华为云IoTDA产品模型规范、腾讯云物联网开发平台物模型协议
>
> 本文档定义建筑机电设备物模型的三维结构（属性、事件、服务），以及设备出厂额定特征的数据规范

---

## 一、物模型总体框架

### 1.1 三维结构定义

| 维度 | 标识 | 说明 | 触发方式 |
|------|------|------|----------|
| 属性（Property） | property | 描述设备运行状态、配置参数、额定参数的数据项 | 设备上报 / 平台下发设置 |
| 事件（Event） | event | 设备主动上报的通知信息，含信息、告警、故障三类 | 设备主动上报 |
| 服务（Service） | service | 平台可向设备下发的指令或可调用的动作 | 平台调用 |

### 1.2 数据类型体系

参照阿里云TSL + 华为云IoTDA，统一数据类型定义：

| 数据类型 | 标识 | 说明 | 规范字段 |
|----------|------|------|----------|
| int | int | 32位整数 | min, max, step, unit |
| float | float | 单精度浮点数 | min, max, step, unit |
| double | double | 双精度浮点数 | min, max, step, unit |
| bool | bool | 布尔值，底层0/1 | 0的值描述, 1的值描述 |
| enum | enum | 枚举整数值 | 枚举项列表（值→描述） |
| text | text | 字符串 | length（最大长度） |
| date | date | UTC毫秒时间戳字符串 | — |
| struct | struct | 结构体，嵌套上述7种类型 | 各子字段定义 |
| array | array | 数组，元素可为int/float/double/text/struct | size（最大长度）, 元素类型 |

### 1.3 属性访问模式

| 模式 | 标识 | 说明 |
|------|------|------|
| 只读 | r | 设备上报，平台不可设置（如测量值、额定参数） |
| 读写 | rw | 平台可下发设置，设备也可上报（如设定温度、运行模式） |

### 1.4 事件类型

| 事件级别 | 标识 | 说明 | 示例 |
|----------|------|------|------|
| 信息 | info | 设备运行信息通知 | 滤网清洗提醒、定时任务完成 |
| 告警 | alert | 设备运行异常预警，可自动恢复 | 温度偏高告警、水压偏低告警 |
| 故障 | error | 设备故障，需人工干预 | 压缩机故障、传感器断路 |

### 1.5 服务调用模式

| 模式 | 标识 | 说明 |
|------|------|------|
| 异步 | async | 平台下发指令后不等待设备响应 |
| 同步 | sync | 平台下发指令后等待设备返回执行结果 |

### 1.6 属性分类

| 类别 | 标识 | 说明 | 访问模式 | 示例 |
|------|------|------|----------|------|
| 运行状态 | runtime | 设备当前运行状态 | r | 开关状态、运行模式、故障代码 |
| 测量参数 | measurement | 传感器实时测量值 | r | 温度、压力、流量、湿度 |
| 设定参数 | setting | 可配置的运行参数 | rw | 设定温度、设定压力、时间计划 |
| 额定参数 | rated | 出厂铭牌参数，运行时不变 | r | 额定功率、额定电压、额定流量 |
| 统计参数 | statistic | 累计统计数据 | r | 累计运行时间、累计能耗、启停次数 |
| 环境参数 | environment | 设备运行环境参数 | r | 环境温度、环境湿度 |
