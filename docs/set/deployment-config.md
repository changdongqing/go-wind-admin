# 部署后配置说明（面向实施交付人员）

> 适用对象：**实施 / 交付 / 运维人员**（无需开发背景）
>
> 本文档说明如何在**不重新编译、不联系开发人员**的前提下，修改 GoWind Admin 前端的以下外观配置：
>
> 1. **站点 / 应用名称**（浏览器页签标题 + 后台左上角标题）
> 2. **Logo 图片**（左上角标题旁的小图标）
> 3. **浏览器页签图标**（favicon，浏览器标签页上的小图标）
> 4. **登录页品牌标题与描述**（登录页左侧的大标题和说明文字）
> 5. **版权信息**（登录页底部及后台页脚的公司名、年份、备案号等）

---

| 版权·公司名 `copyrightCompanyName` | 编辑 `app-config.js` | 登录页底部、后台页脚 |

## 一、核心概念：什么是「部署目录」？

前端打包后会生成一个产物目录（通常叫 `dist/`），把这个目录部署到服务器（如 Nginx 的静态资源根目录）后，实施人员要修改的就是**这个目录里的文件**，而不是源码。

本文后续提到的"部署目录"指的就是它。下文涉及的文件都在该目录的**根层**：

```
部署目录/
├── index.html          ← 系统入口页面（不要改）
├── app-config.js       ← 【改这个】站点名称、登录页品牌文案、版权信息
├── logo.png            ← 【换这个】Logo 图片
├── favicon.ico         ← 【换这个】浏览器页签图标
└── assets/             ← 系统其他资源（不要改）
```

> 配置方式分两类：
> - **编辑文件内容**：站点名称、登录页品牌文案、版权信息 → 编辑 `app-config.js`
> - **替换同名文件**：Logo、页签图标 → 用新图片覆盖 `logo.png` / `favicon.ico`
>
> ⚠️ 文件名必须**完全一致**（包括大小写、后缀），改名就不会生效。

## 二、修改「站点 / 应用名称」

### 作用
- 浏览器标签页上显示的标题
- 后台界面左上角显示的应用名称

### 操作步骤
1. 用**记事本**或 **VS Code** 打开部署目录下的 `app-config.js`。
2. 找到这一行：
   ```js
   appName: "GoWind Admin",
   ```
3. 把双引号里的文字改成你想要的名称，例如：
   ```js
   appName: "风行管理平台",
   ```
4. 保存文件。

### 注意事项
- ✅ 只改双引号**里面的文字**，不要删除双引号、冒号、逗号。
- ❌ 不要改成空字符串 `""`，否则标题会空白。

### 生效方式
刷新浏览器即可（如有缓存，按 `Ctrl + F5` 强制刷新）。

---

## 三、修改「Logo 图片」

### 作用
后台界面左上角标题旁边显示的小图标。

### 操作步骤
1. 准备一张新的 Logo 图片：
   - 格式：**PNG**（建议透明背景）
   - 形状：**正方形**（系统会自动缩放到约 32×32 像素显示）
   - 建议原始尺寸：**≥ 64×64 像素**（保证高清屏清晰）
   - 文件名必须命名为：**`logo.png`**
2. 用新的 `logo.png` **替换**部署目录下的同名旧文件。

### 注意事项
- ⚠️ 文件名必须是 `logo.png`（全小写，.png 后缀）。用 `.jpg`、`.svg` 或改名都不生效。
- ⚠️ 替换时选择"覆盖 / 替换目标文件"，不要变成 `logo (1).png`。

### 生效方式
刷新浏览器（图片有缓存，按 `Ctrl + F5` 强制刷新）。

---

## 四、修改「浏览器页签图标」（favicon）

### 作用
浏览器标签页、收藏夹、地址栏里显示的那个小图标。

### 操作步骤
1. 准备一张新的图标文件：
   - 格式：**ICO**（推荐）或多尺寸 PNG
   - 推荐尺寸：包含 **16×16** 和 **32×32** 两种（ICO 格式可在一个文件内包含多种尺寸）
   - 文件名必须命名为：**`favicon.ico`**
   - 可使用在线工具（搜索"favicon 在线生成"）把 PNG 转成 ICO。
2. 用新的 `favicon.ico` **替换**部署目录下的同名旧文件。

### 注意事项
- ⚠️ 文件名必须是 `favicon.ico`。
- ⚠️ favicon 的浏览器缓存非常顽固，替换后**必须**按 `Ctrl + F5` 强制刷新；有时甚至需要关闭浏览器重新打开，或清理浏览器缓存才能看到新图标。

### 生效方式
强制刷新浏览器（`Ctrl + F5`）；顽固缓存需清空浏览器缓存或重启浏览器。

## 五、修改「登录页品牌标题与描述」

### 作用
登录页左侧显示的**大标题**和**说明文字**（例如默认的"风行中后台管理系统"及其下方的描述）。

### 操作步骤
1. 打开 `app-config.js`，找到这两行：
   ```js
   systemTitle: "风行中后台管理系统",
   systemDescription: "开箱即用的企业级中后台管理系统",
   ```
2. 把双引号里的文字改成目标文案，例如：
   ```js
   systemTitle: "有名智能管理平台",
   systemDescription: "企业级中后台管理系统",
   ```
3. 保存文件。

### 注意事项
- ✅ 若不想自定义，把值留空 `""` 即可，系统会使用内置默认文案。
- ℹ️ 此处文案**不区分中英文**：配置后，中文和英文登录页都会显示该文案（品牌名通常不随语言变化）。

### 生效方式
刷新浏览器（`Ctrl + F5`），在**未登录的登录页**查看效果。

## 六、修改「版权信息」

### 作用
登录页底部及后台页脚显示的版权信息，例如 `Copyright © 2026 GoWind`。

### 各字段的显示位置（重要）

不同字段显示在界面的不同位置，配了不一定会立刻"全部"看到：

| 字段 | 登录页底部 | 后台页脚 |
|---|---|---|
| `copyrightCompanyName`（公司名） | ✅ 显示 | ✅ 显示 |
| `copyrightYear`（年份） | ✅ 显示 | ✅ 显示 |
| `copyrightIcp`（备案号） | ❌ 不显示 | ✅ 显示为可点击链接 |
| `copyrightSiteLink`（公司网址） | ❌ 不显示 | ❌ 当前版本未显示（预留） |

> ℹ️ **后台页脚默认是关闭的**。若希望版权信息也出现在后台界面底部，需要在登录后进入**系统设置 → 偏好设置**，开启"显示页脚"选项。开启后，后台每个页面底部都会显示版权信息（含备案号链接）。
>
> ℹ️ 备案号链接点击后跳转的网址，由代码内部的备案链接地址决定（非本文件配置）。如需修改备案号跳转地址，请联系开发人员。

### 操作步骤
1. 打开 `app-config.js`，找到版权相关配置：
   ```js
   copyrightCompanyName: "GoWind",
   copyrightYear: "2026",
   copyrightSiteLink: "https://www.example.com",
   copyrightIcp: "",
   ```
2. 按需修改各项：

   | 字段 | 含义 | 示例 |
   |---|---|---|
   | `copyrightCompanyName` | 版权归属公司名 | `"有名智能科技有限公司"` |
   | `copyrightYear` | 年份或年份范围 | `"2026"` 或 `"2021-2026"` |
   | `copyrightSiteLink` | 公司网址 | `"https://www.example.com"` |
   | `copyrightIcp` | ICP 备案号（仅中国大陆部署需要） | `"沪ICP备XXXXXXXX号"` |

3. 不需要的项保持空字符串 `""` 即可。
4. 保存文件。

### 生效方式
刷新浏览器（`Ctrl + F5`）。

## 七、常见问题（FAQ）

### Q1：改完没生效？
按顺序排查：
1. 是否**强制刷新**了浏览器（`Ctrl + F5`）？
2. 文件名是否**完全一致**（`app-config.js` / `logo.png` / `favicon.ico`，注意大小写）？
3. `app-config.js` 里是否**误删了引号、冒号或逗号**？（这是最常见的错误。可用浏览器打开页面后按 `F12` → Console 面板查看是否有红色报错）
4. 是否改对了目录？确认改的是**服务器上实际部署的目录**，而不是本地源码。

### Q2：`app-config.js` 改坏了怎么办？
该文件内容很简单，若不慎改错，把对应行改回标准写法即可。完整默认示例：
```js
window.__APP_CONFIG__ = {
  appName: "GoWind Admin",
  systemTitle: "风行中后台管理系统",
  systemDescription: "开箱即用的企业级中后台管理系统",
  copyrightCompanyName: "GoWind",
  copyrightYear: "2026",
  copyrightSiteLink: "https://www.example.com",
  copyrightIcp: "",
};
document.title = window.__APP_CONFIG__.appName;
```

### Q3：可以改其他东西吗（比如主题色、登录按钮文字）？
当前版本开放了以下免编译配置：**站点名称、Logo、页签图标、登录页品牌文案、版权信息**。如需修改主题色、按钮文字、功能逻辑等，请联系开发人员调整后重新打包。

### Q4：换一个客户部署，需要重新打包吗？
**不需要**。把同一份打包产物复制到新客户的服务器，然后按本文档修改 `app-config.js`、替换 `logo.png` 和 `favicon.ico` 即可完成客户化。

部署到新环境后，逐项确认：

- [ ] `app-config.js`：`appName` 已改为目标站点名称
- [ ] `app-config.js`：`systemTitle` / `systemDescription` 已改为目标登录页文案
- [ ] `app-config.js`：版权信息（公司名 / 年份 / 备案号）已改
- [ ] `logo.png` 已替换为新 Logo
- [ ] `favicon.ico` 已替换为新图标
- [ ] 浏览器强制刷新（`Ctrl + F5`）后，各项显示均符合预期
