# 建筑系统分类清单

> 依据标准：GB/T 51269-2017《建筑信息模型分类和编码标准》、DB31/T 1320-2021《公共机构建筑设备设施数据编码规则》、T/BCAT 0002-2022《综合建筑与设备设施编码标准》
>
> 分类层级：大类 → 中类 → 小类 → 细类（四层，无第五层）
>
> 编码规则：大类2位 + 中类2位 + 小类2位 + 细类2位 = 8位编码
>
> 说明：本清单中的"系统"指由设备、管道、线缆等组成的具有独立功能的整体系统，而非单个设备

## 分类说明

### 大类划分逻辑

| 大类 | 含义 | 依据 |
|------|------|------|
| 暖通空调系统 | 供暖、通风、空调及冷热媒输配的完整系统 | DB31/T 1320 表C.1-C.2子系统 |
| 给排水系统 | 给水、排水、热水、消防给水的完整系统 | DB31/T 1320 表C.1子系统 |
| 电气系统 | 变配电、备用电源、照明、防雷接地的完整系统 | DB31/T 1320 表C.1子系统 |
| 消防系统 | 火灾报警、灭火、防排烟、联动的完整系统 | DB31/T 1320 表C.1子系统 |
| 电梯系统 | 电梯、自动扶梯的运行与监控系统 | GB/T 51269 表30 |
| 智能化系统 | 通信、安防、楼控、音视频、集成的完整系统 | GB/T 51269 表31-36 |
| 建筑围护与结构系统 | 围护、结构、防水、保温的完整系统 | GB/T 51269 表21-22 |

### 合并原则

- 系统与设备的区别：系统关注"由设备+管道/线缆+控制组成的整体功能"，设备关注"单体设备"
- 消防给水系统归入"给排水系统"大类（与水系统逻辑一致），消防报警和灭火控制归入"消防系统"大类
- 暖通空调的冷热媒输配系统（冷冻水、热水、蒸汽、冷却水）统一归入暖通空调系统，不再独立为管道系统
- 建筑围护与结构系统合并围护、结构、防水、保温，避免按构件类型过度拆分
- 智能化系统将通信、安防、楼控、音视频、集成平台合并为一个大类，与设备分类保持对应
